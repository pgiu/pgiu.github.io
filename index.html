Personal projects

Amazon Alexa Soccer App

As a soccer fan and an Alexa enthusiast I decided to build my own app to get the latest updates from my favorite team. It's a serverless application with two parts: a back-end to pull the data, and the Alexa VUI (Voice User Interface) to interact with the data. The Alexa VUI can answer questions like: 'what's my team update?' or 'When is X playing Y?'. The back-end is a javascript function that scrapes a soccer stats website, to populate a DynamoDB database.

Take a look here!

Technologies: Amazon Lambda, DynamoDB, Alexa, JavaScript

GTFS Merger

The General Transit Feed Specification (GTFS) is a file format promoted by Google, that defines a common way for public agencies to share information about transportation schedules and associated geographic information. The script I wrote is a tool that was used to make data consistent with pre-existent scripts, used to analyze transportation accesibility in different South American cities.

Technologies: Python
